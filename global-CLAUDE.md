## 回复偏好

- 始终用中文回复
- 回答要简洁直接
- 不要使用 emoji
- 不要在每次操作后总结你做了什么，我能看到

## 通用安全规则

- 绝对不要提交 .env、API Key、密码、token 到 git
- 未经我确认，不要执行破坏性操作（rm -rf、force push、reset --hard）
- 不要自作主张添加新的依赖包，先问我

## 通用工作习惯

- 修改代码前先读代码，理解上下文
- 保持简单，不要过度设计
- 只做我要求的事情，不要额外添加功能或重构
